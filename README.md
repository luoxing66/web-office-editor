# Web Office Editor

一个轻量级的 Windows 桌面应用，通过自定义 URL Scheme 协议，让任意 Web 系统（OA/ERP/CRM）快速集成 Office 文件的本地编辑能力。

## 🧪 在线 Demo
我们提供了一个基于 Nuxt 的在线演示站点：

🔗 https://office-editor.tuojiecloud.cn

## ✨ 核心特性
- 📦 极轻量：安装包仅几百 KB，无依赖、无后台服务
- 🔗 协议唤起：通过 office-editor:// 协议从浏览器直接打开本地 Office 应用
- 🧰 灵活集成：后端只需实现两个标准 HTTP 接口，适用于任何技术栈
- 🔐 数据自主可控：文件流由业务系统控制，无需上传至第三方服务

## 🧩 适用场景
- 企业 OA、ERP、CRM 等系统嵌入 Office 编辑能力
- 政务、金融、医疗、教育等数据敏感行业
- 私有化部署、内网环境、合规要求严格的业务系统
- 在线方案渲染效果不佳、交互体验不理想

## 🛡️ 安全与隐私
- 所有文件数据均直接与您的后端交互，本应用不存储任何用户数据
- 支持内网系统，不依赖公网服务
